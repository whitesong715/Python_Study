form 태그

form 태그는 서버에 데이터를 전송해주는 기능을 하는 태그이다.
주요 속성 : method, action
method : 데이터를 전송할 방식을 결정한다. get과 post로 나뉜다.
action : 요청할 주소를 작성한다.

get방식으로 데이터 전송하기

post방식으로 데이터 전송하기

form 태그의 method속성은 반드시 필요한 속성으로 default값을 가지고 있다.
default값 : get

사용자로부터 데이터를 입력받기 위한 input태그

input태그는 사용자가 입력할 수 있는 화면을 출력하는 태그.
입력화면은 type속성에 따라 결정된다.
input태그는 반드시 form태그에 있어야하는것은 아니다.

type : text, password

text, password타입은 한 줄 문자를 입력받는 화면

type : hidden

type : button

버튼 모양을 출력 -> 반드시 기능(이벤트 처리)을 추가해야한다 -> 자바스크립트와 연결해서 사용

type : checkbox

checkbox 타입은 다중선택 가능
선택사항을 두고 체크, 선책할 수 있는 화면을 출력
name/value값을 반드시 지정해야한다.
동일한 선택조건(카테고리)에 있을 경우 name을 동일하게 설정한다. ( 서버에서 데이터를 배열형식으로 받아오기 때문 )

type : file

파일을 서버에 업로드할 때 사용 ( local에 있는 파일을 가져올 때 사용 )
바이너리 파일이 전송되기 때문에 form으로 전송할 때는 encoding값을 변경해야한다. : enctype 속성 이용

type : image

image 타입은 사진을 submit으로 이용하는 타입

type : submit, reset

submit : form 태그의 자식인 input태그의 값을 전송하는 태그 reset : form 태그의 자식인 input태그의 값을 초기화하는 태그 submit, reset은 버튼으로 출력되고, 버튼에 출력되는 내용은 value속성으로 설정된다.

type : search, range, url, email

search : 검색어 입력창, text타입과 유사하나 x버튼이 있다 ( 전체 지우는 용)
range : 일정 범위에 있는 값을 표시, 속성에 max와 min, step값이 필요하다
url : 사이트의 주소를 입력받는 입력창, text타입과 유사하나 입력값에 'http://' 문구가 있는지 체크한다
email : 이메일을 입력받는 입력창, text타입과 유사하나 '@'가 있는지 ( 이메일 형식에 맞는지 ) 체크한다





기타 입력 창 태그들

select : 콤보박스 형식으로 선택하는 것 ( 예시 : 검색 카테고리 ), 자식태그로 option태그(value값)를 가지고 있다
detailist : input태그 text타입의 자동완성기능을 할 수 있게 하는 태그, 자식태그로 option태그(value값)를 가지고 있다
textarea : 여러문장을 입력받게 하는 태그, 긴글이나 소개글, 게시판 내용등에 쓰인다. 특정크기를 속성으로 정해야한다. ( cols, rows)

type : select, datalist, textarea
